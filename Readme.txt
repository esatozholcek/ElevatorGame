How To Play The Game ?

Move the Elevator between 0-10 Floors with "UP" and "DOWN" buttons.
When the person reach the target floors, Person will leave.
If you want to add a person from current floor, please enter the name of person and target of person.

You can see the statistics like :
-Elevator current floor
-Number of reached people
-Travel meter
-Elavator is empty / Name of last person
